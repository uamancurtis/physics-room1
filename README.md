# 跨学科项目学习智能评估系统

## 系统功能

### 1. 用户角色与材料上传
- 8个小组和教师角色
- 小组可上传项目报告（支持txt/doc/docx/pdf格式）和实验视频（MP4格式，≤100M）
- 所有用户均可查看所有小组上传的材料

### 2. 多维度评分系统
- **AI评分（20%）**：通过模拟AI大模型对项目报告进行5个维度评分
- **教师评分（30%）**：教师对各小组进行综合评分
- **小组自评（10%）**：小组对自己的作品进行评分
- **小组互评（40%）**：小组间相互评分
- 评分维度：项目创新性、技术实现、实验设计、成果展示、团队合作

### 3. 实时排名与奖状
- 动态显示各小组得分排名
- 前三名小组可申请电子奖状
- 奖状包含名次、组号、时间等信息

### 4. 数据同步
- 使用localStorage实现数据持久化
- 不同终端登录可查看一致数据

## 技术实现

### 前端技术栈
- **HTML5**：页面结构
- **CSS3**：样式设计，响应式布局
- **JavaScript (ES6+)**：交互逻辑与数据处理

### 核心功能实现
- 用户认证与角色管理
- 材料上传与展示
- 多维度评分系统
- 实时排名计算
- 电子奖状生成
- 本地数据持久化

## 使用方法

### 1. 启动系统
直接在浏览器中打开`index.html`文件即可使用。

### 2. 登录系统
- 选择角色（小组/教师）
- 小组角色需选择组号（1-8）
- 点击登录进入系统

### 3. 上传材料
- 进入"上传材料"页面
- 选择项目报告和实验视频
- 点击"上传"按钮完成提交

### 4. 查看材料
- 进入"查看材料"页面
- 可选择查看所有小组或特定小组的材料
- 视频可直接在页面上播放

### 5. 评分
- 进入"评分"页面
- 小组角色：可进行自评和互评
- 教师角色：可进行教师评分和生成AI评分
- 拖动滑块进行评分，点击保存按钮

### 6. 查看排名
- 进入"排名"页面
- 查看实时动态排名
- 前三名小组可点击"申请奖状"按钮生成电子奖状

## 浏览器兼容性

支持主流现代浏览器：
- Chrome (推荐)
- Firefox
- Safari
- Edge

## 注意事项

1. 由于是纯前端实现，数据存储在浏览器的localStorage中，若清除浏览器缓存，数据将丢失
2. 文件上传功能仅在界面上模拟，实际文件不会被真正上传
3. AI评分功能使用模拟数据，实际部署时需接入AI大模型API
4. 视频播放功能需要实际视频文件支持
